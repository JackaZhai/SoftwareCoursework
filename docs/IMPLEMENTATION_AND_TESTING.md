# 高校毕业生事业发展跟踪管理系统实现与测试说明

## 系统概览
- **技术栈**：Flask + Bootstrap，Python 以模块方式划分七大子系统，HTML 模板用于可视化演示。
- **存储实现**：使用 `app.storage.InMemoryStorage` 作为内存数据库，包含用户、毕业生、问卷、通知、报表与审计日志。
- **路由划分**：
  - `/auth` 身份与权限：用户管理、MFA 标记、SSO 占位提示。
  - `/graduates` 毕业生信息：基础档案、就业状态、历史与附件占位。
  - `/surveys` 问卷与调查：问卷模板、版本、答卷收集。
  - `/analytics` 统计分析：就业分布与问卷参与度指标示例。
  - `/reports` 报表与文档：生成报告草稿、审批归档。
  - `/notifications` 消息推送：短信/邮件/微信发送记录。
  - `/audit` 日志审计与监控：展示操作日志，预留对接 ELK/Prometheus。

## 运行步骤
1. 安装依赖：`pip install flask`。
2. 启动应用：`python run.py`，浏览器访问 `http://localhost:5000`。
3. 通过导航栏进入各子系统进行演示操作。
